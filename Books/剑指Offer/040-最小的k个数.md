# 040-最小的k个数

## 题目

```
输入整数数组 arr ，找出其中最小的 k 个数。例如，输入4、5、1、6、2、7、3、8 这8个数字，则最小的4个数字是1、2、3、4。

输入：arr = [3,2,1], k = 2
输出：[1,2] 或者 [2,1]

限制：
0 <= k <= arr.length <= 10000
0 <= arr[i] <= 10000
```

<br>

## 解法 


### Python

```
## Time: O(nlogn)
def getLeastNumbers_sort(self, arr: list, k: int) -> list:
	arr.sort()
	return arr[:k]
```
